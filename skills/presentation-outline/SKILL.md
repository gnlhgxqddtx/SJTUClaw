# Presentation Outline Skill

本 skill 用于把主题与材料转化为条理清晰、便于开讲的展示大纲。

## 适用场景
- 课堂汇报 / 组会汇报的结构设计
- 演讲讲稿提纲
- PPT 页面（分页）结构规划

## 工作流程
1. 明确要素：主题、时长或页数、听众、可用材料（如在 workspace 中先 `read_file` 读取）。
2. 设计整体叙事线：开场 → 背景/问题 → 核心内容（2-4 部分）→ 结论 → Q&A/致谢。
3. 为每一部分/每一页给出：页标题、要点（bullet）、可选的讲稿提示（speaker notes）。
4. 控制信息密度：每页 3-5 个要点，避免整段文字。
5. 若用户要求保存，通过 update tool 写入指定路径并经 approval。

## 输出要求
- 用 Markdown，按"页/节"组织，形如 `## 第 N 页：标题` + 要点列表 + （可选）讲稿提示。
- 给出建议时长分配（若用户提供了总时长）。
- 结尾附一句总体讲述建议或注意事项。
