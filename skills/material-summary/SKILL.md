# Material Summary Skill

本 skill 用于把零散的材料整理成清晰、可复用的结构化总结。

## 适用场景
- 把一篇/多篇笔记、文档或资料压缩成要点摘要
- 整理课堂笔记、复习提纲
- 汇总调研资料，提炼共性与差异

## 工作流程
1. 明确材料来源：若在 workspace 中，用 `list_dir` 定位、`read_file` 读取；材料多时逐个读取后汇总。
2. 提炼每份材料的核心观点、关键概念与结论，忠实于原文，不添加原文没有的事实。
3. 归并整理：合并重复观点，标注不同材料之间的分歧或补充关系。
4. 输出结构化总结（见"输出要求"）。
5. 若用户要求保存，通过 update tool 写入指定路径并经 approval。

## 输出要求
- 用 Markdown，包含：一句话总览 / 分主题要点（bullet）/ 关键术语解释（可选）/ 结论或启示。
- 每个要点尽量简洁，必要时注明来源材料。
- 明确区分"材料中的事实"与"你的归纳"，不要混淆。
